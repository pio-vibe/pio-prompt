# pio-prompt 🤖

AI Prompt Builder - Build perfect prompts like building blocks

## Features
- 40+ Prompt Cards
- Mix & Match
- Multiple Categories (Productivity, Content, Learning, Business, Developer, Personal)
- Dark Mode UI

## Quick Start
```bash
npm install
npm run dev
```

## Tech Stack
- Next.js 14
- Tailwind CSS
- Supabase (optional)
- Stripe (optional)
